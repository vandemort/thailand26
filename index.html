Hướng Dẫn Chia Sẻ Cho Cả Đoàn
Ví dụ thực tế: Bưởi vừa cập nhật nhà Huyền đã nộp đủ 45M, và thêm khách sạn mới vào lịch trình. Bây giờ Bưởi muốn gửi bản mới nhất cho cả 8 người trong đoàn.
Trên điện thoại (Android / iPhone)
1
Nhấn nút "Xuất" ở góc trên phải. Điện thoại sẽ tải về file .json (thường nằm trong thư mục Tải về / Downloads).
2
Mở Zalo → vào nhóm chat của đoàn → nhấn biểu tượng đính kèm file → tìm và chọn file vừa tải → Gửi.
3
Người nhận (ví dụ nhà Huệ) nhấn giữ file trong Zalo → chọn "Tải về" → sau đó mở ứng dụng này → vào tab Gia Đình → cuộn xuống cuối → nhấn "Nhập dữ liệu từ file JSON" → chọn file vừa tải. Xong!
Trên máy tính (laptop / PC)
1
Nhấn nút "Xuất" → file tự tải về thư mục Downloads trên máy tính.
2
Mở Zalo Web (zalo.me) hoặc Facebook Messenger trên trình duyệt → vào nhóm → đính kèm file → gửi.
3
Người nhận trên máy tính: Tải file về → mở ứng dụng này trên trình duyệt → tab Gia Đình → "Nhập dữ liệu" → chọn file. Dữ liệu mới hiện ra ngay.
Lưu như app trên màn hình chính
A
iPhone/iPad: Mở file trong Safari → nhấn biểu tượng Chia sẻ (hình vuông + mũi tên) → chọn "Thêm vào Màn hình chính" → nhấn Thêm. Sau đó mở như app bình thường.
B
Android: Mở file trong Chrome → nhấn dấu 3 chấm (góc phải trên) → chọn "Thêm vào màn hình chính" → Thêm. Icon xuất hiện như app.
Lưu ý quan trọng: Mỗi người lưu dữ liệu riêng trên máy của mình. Khi có thay đổi (thêm chi tiêu, cập nhật lịch...), người quản lý chỉ cần Xuất → Gửi lại cho cả nhóm. Ai nhận thì nhấn Nhập để cập nhật bản của mình.

Tiếng Thái Thông Dụng
Giao tiếp cơ bản
Xin chào → Sawasdee kha/krub (สวัสดีค่ะ/ครับ) — nữ dùng "kha", nam dùng "krub"
Cảm ơn → Khob khun kha/krub (ขอบคุณ)
Xin lỗi / Thứ lỗi → Kho thot (ขอโทษ)
Có / Không có → Mee / Mai mee (มี / ไม่มี)
Được / Không được → Dai / Mai dai (ได้ / ไม่ได้)
Tôi không hiểu → Mai khao jai (ไม่เข้าใจ)
Ăn uống
Ngon lắm! → Aroy mak! (อร่อยมาก)
Không cay → Mai phet (ไม่เผ็ด)
Cay vừa thôi → Phet nit noi (เผ็ดนิดหน่อย)
Tính tiền → Kep ngern duay (เก็บเงินด้วย)
Nước lọc → Nam plao (น้ำเปล่า)
Không lấy túi nilon → Mai ao tung (ไม่เอาถุง)
Mua sắm & đi lại
Bao nhiêu tiền? → Tao rai? (เท่าไหร่)
Đắt quá, giảm được không? → Phaeng pai, lot noi dai mai? (แพงไป ลดหน่อยได้ไหม)
Nhà vệ sinh đâu? → Hong nam yoo tee nai? (ห้องน้ำอยู่ที่ไหน)
Đi đến... được không? → Pai... dai mai? (ไป...ได้ไหม)
Dừng ở đây → Yoot tee nee (หยุดที่นี่)
